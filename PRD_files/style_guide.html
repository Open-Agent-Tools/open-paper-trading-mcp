Open Paper Trading Style Guide

Design system inspired by Dieter Rams and Massimo Vignelli principles of clarity, simplicity, and functional beauty. This guide ensures color-blind accessibility and provides comprehensive UI components for consistent user experience.

Design Principles

Dieter Rams - Good Design

  • Is innovative
  • Makes a product useful
  • Is aesthetic
  • Makes a product understandable
  • Is unobtrusive
  • Is honest
  • Is long-lasting
  • Is thorough down to the last detail
  • Is environmentally friendly
  • Is as little design as possible

Massimo Vignelli - Design Fundamentals

  • Semantics - meaning and context
  • Syntactics - visual relationships
  • Pragmatics - functional purpose
  • Typography as primary design element
  • Grid systems for organization
  • Appropriate materials and colors
  • Timeless over trendy
  • Discipline and consistency

Color Accessibility

All color combinations meet WCAG 2.1 AA standards for contrast ratio (4.5:1 minimum). The palette has been tested with Deuteranopia and Protanopia simulators to ensure accessibility for color-blind users.

Usage Guidelines

Implementation Notes

  • Use CSS custom properties (variables) for consistent color application
  • Always test color combinations with color-blind simulators
  • Maintain minimum 4.5:1 contrast ratio for text
  • Use semantic HTML elements for better accessibility
  • Implement focus indicators for all interactive elements
  • Test responsiveness across device sizes
CSS Custom Properties Usage
/* Use variables for consistency */
.my-component {
  background-color: var(--primary-blue);
  color: var(--neutral-white);
  padding: var(--space-md);
  border-radius: var(--radius-md);
}
